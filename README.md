# Aplikasi Laundry

Aplikasi manajemen transaksi laundry berbasis Java Desktop, dibuat menggunakan NetBeans dan database MySQL.

## Fitur
- Input transaksi (jenis cuci, harga/kg, berat, total harga, bayar)
- Data laundry (riwayat transaksi)
- Update & hapus data

## Teknologi
- Java (Swing)
- NetBeans IDE
- MySQL
- Maven (pom.xml)
